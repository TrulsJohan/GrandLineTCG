## Project Description

This is a console-based **Tournament Management Platform** developed in C#. 
The application allows users to register, log in, 
create and manage events, browse and search for events, 
book tickets, and leave reviews after attending.

We chose the theme **"GrandLineTCG"** – a platform focused on **Trading Card Games** in the Oslo area.
This domain fits naturally with object-oriented design and allows
for clear differentiation between event types.

### Key Features
- User registration and authentication
- Create, edit, and cancel events (as organizer)
- Browse, filter, and search upcoming events
- Multiple ticket/booking types with limited capacity
- Booking and cancellation system
- Post-event reviews with average rating calculation
- User profile with organized events and booking history
- Proper access control and input validation

## C# Features Demonstrated

- **Generic Collections**  
  Used to store and manage groups of objects in a type-safe way, making the code more flexible and reducing runtime errors.

- **LINQ**  
  Used to efficiently query and manipulate collections (such as filtering, sorting, and selecting data) with clean and readable syntax.

- **Enums**  
  Used to represent fixed sets of related constants (e.g., card types or states), improving code clarity and reducing the use of “magic numbers” or strings.

- **Exception Handling**  
  Used to handle runtime errors, preventing crashes and allowing the application to respond to unexpected situations.

- **Class Inheritance**  
  Used to create a shared structure between related classes, promoting code reuse and making it easier to extend functionality.
